What is Responsive Design

In order to understand responsive design we need to first define other approaches / strategies used in web design, in fact responsive design at the basic level is a combination of fluid layouts and adaptive layouts.

Fluid layouts leverage the ability of CSS to specify the size of elements in percentages and other relative measures (em). The advantage of a fluid layout is that the elements on the page automatically adjust as the vieport changes.

Adaptive layout use mainly @media queries as standardized in CSS3 to apply different styles based on the various conditions. The most common condition to use is to change styles based on viewport width.
With adaptive layout we could for example hide an element on mobile devices or change its position.

When used skillfully the combination of fluid and adaptive layouts can modify the user experience to make it optimal on all devices.
